# BlackJack Game

## Overview
A virtual BlackJack card game built with Python and Pygame. Players compete against a dealer, starting with $1500 to bet. The game features standard BlackJack mechanics including Hit, Stand, and betting.

## Project Structure
- `game.py` - Main game file containing all game logic and Pygame UI
- `README.md` - Original project documentation
- `BLOCKERS.md` - Development blockers log
- `STANDUP_NOTES.md` - Team standup notes

## Technology Stack
- Python 3.12
- Pygame 2.6.1

## How to Run
Run the project using the "BlackJack Game" workflow, which executes `python game.py`. The game displays in a VNC window.

## Game Features
- Standard BlackJack rules with dealer standing on 17
- Betting system with adjustable bet amounts
- Visual card display with suit symbols
- Win/Lose/Push detection
- Money tracking across rounds
